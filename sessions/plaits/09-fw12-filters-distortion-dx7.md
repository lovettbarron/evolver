# Session 09: Firmware 1.2 — Filters, Phase Distortion & DX7 FM

**Objective:** Explore the first three Orange bank models added in firmware 1.2: classic waveshapes with filter, phase distortion & modulation, and the 2-voice 6-operator FM synth with 32 DX7-style presets.

> [!tip] If you only have 5 minutes
> Navigate to Orange bank (press both MODEL buttons until LEDs turn orange). Select Model 3. Sweep HARMONICS slowly -- you are cycling through 32 DX7-style FM presets, each with a completely different character. This is a 6-operator FM synth inside a 12HP module.

## Accessing the Orange Bank

Firmware 1.2 adds a third bank (Orange LEDs) alongside the original Green and Red banks. To access it:
1. Press both MODEL buttons simultaneously to cycle banks: Green → Red → Orange → Green
2. Or: hold one MODEL button and press the other to advance banks
3. The LED column color indicates the active bank: green, red, or orange

## Warm-Up (2 min)

Set init state. Cycle through all three banks to confirm firmware 1.2 is installed -- you should see green, red, and orange LED colors. If you only see green and red, your Plaits needs a firmware update.

## Setup

1. Set all controls to init state
2. Patch V/OCT to **V/OCT**, OUT to mixer
3. Navigate to Orange bank, Model 1

## Exercises

### Exercise 1: Classic Waveshapes with Filter (7 min)

Orange Model 1 pairs classic waveforms with a built-in filter -- giving Plaits a complete subtractive voice:

1. Play a note. Sweep **TIMBRE**: this is the filter cutoff. Hear the brightness change from dark to open
2. Sweep **HARMONICS**: this controls filter character and resonance. CCW = gentle 24dB/oct, CW = harsh 12dB/oct
3. Sweep **MORPH**: this changes the waveform and sub-oscillator level -- triangle through saw through square with growing sub content
4. **OUT** carries the LP (low-pass) output. **AUX** carries a 12dB/octave HP (high-pass) output -- patch both for a complete filter pair
5. Try patching a CV source to the FM input with the FM attenuverter turned up -- this creates filter sweep modulation

### Exercise 2: Phase Distortion & Modulation (7 min)

<div data-plaits-panel
  data-knobs="knob-plaits-frequency:64,knob-plaits-harmonics:64,knob-plaits-timbre:64,knob-plaits-morph:64"
  data-highlights="knob-plaits-harmonics:amber,knob-plaits-timbre:blue,knob-plaits-morph:blue"
></div>

Switch to Orange Model 2. This is a phase distortion engine inspired by the Casio CZ series:

1. Start with all knobs at noon. Sweep **HARMONICS**: this is the distortion frequency -- the rate at which the phase gets bent
2. Sweep **TIMBRE**: distortion amount. Low = clean sine, high = complex harmonics generated by phase warping
3. Sweep **MORPH**: distortion asymmetry. This changes the character from symmetrical to lopsided distortion
4. **OUT** carries a sync'd carrier (phase distortion). **AUX** carries a free-running carrier (phase modulation) -- these are subtly different synthesis techniques
5. Compare this to Orange Model 1: where Model 1 filters harmonics away (subtractive), Model 2 generates harmonics through phase warping (distortion synthesis)

### Exercise 3: 2-Voice 6-Operator FM Synth — DX7 Presets (9 min)

<div data-plaits-panel
  data-knobs="knob-plaits-frequency:64,knob-plaits-harmonics:0,knob-plaits-timbre:64,knob-plaits-morph:64"
  data-highlights="knob-plaits-harmonics:amber,knob-plaits-timbre:blue,knob-plaits-morph:blue,jack-plaits-trig:amber"
></div>

Switch to Orange Model 3. This is the headline addition of firmware 1.2 -- a complete DX7-style 6-operator FM synthesizer with 32 factory presets loaded via SysEx:

1. Sweep **HARMONICS** very slowly from left to right: each of the 32 positions loads a different DX7 preset. Listen to the dramatic timbral changes -- electric pianos, bells, brass, strings, organs, and more
2. **TIMBRE** controls the modulator level -- this is like the FM "brightness" or modulation index. Low = clean fundamental, high = complex FM harmonics
3. **MORPH** controls the envelope and modulation stretching/time-travel. This morphs through the DX7 envelope stages
4. Patch **TRIG**: the two voices are alternately triggered. Without TRIG patched, a single voice plays as a drone
5. Listen to **OUT** vs **AUX**: OUT is the carrier with sync'd phase (phase distortion), AUX is the carrier with free-running phase (phase modulation) -- different flavors of the same preset
6. Find 3 presets you like. Note the HARMONICS position for each (count clicks or estimate the knob position)
