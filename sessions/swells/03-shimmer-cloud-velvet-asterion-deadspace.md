# Session 03: Ethereal Textures -- Velvet, Asterion & Deadspace

**Objective:** Explore the shimmer and cloud algorithms (Velvet, Asterion, Deadspace), then create a layered ethereal texture patch.

> [!tip] If you only have 5 minutes
> Select Velvet. Set DECAY to 3 o'clock, SIZE to 2 o'clock, MIX to fully wet. Play a single sustained note and let the shimmer build. This is the sound of Swells at its most ethereal.

## Warm-Up (2 min)

Init state. Select Fog. Play audio and sweep DECAY to reconnect with the controls. Then switch to Velvet to enter today's territory.

## Setup

1. Init state (see [Swells Overview](/modules/swells/))
2. Audio source to **IN L**, **L** and **R** outputs to mixer (stereo monitoring recommended)
3. DECAY at **2 o'clock**, SIZE at **2 o'clock**, MIX at **2 o'clock** (slightly wet-biased)
4. Start with **Velvet** algorithm

## Exercises

### Exercise 1: Velvet -- Lush Shimmer (5 min)

<div data-swells-panel
  data-highlights="slider-swells-decay:amber,slider-swells-size:amber,slider-swells-ebb:blue,slider-swells-flow:blue"
></div>

1. Play sustained audio through Velvet. Notice the harmonic shimmer that builds in the tail
2. Increase **DECAY** to 3 o'clock -- the shimmer sustains longer and builds density
3. Sweep **EBB** -- in Velvet mode, this shapes the shimmer character
4. Sweep **FLOW** -- notice how it adds movement to the shimmer texture
5. Try sending short percussive sounds -- Velvet turns them into shimmering cascades

### Exercise 2: Asterion -- Spectral and Otherworldly (5 min)

<div data-swells-panel
  data-highlights="slider-swells-ebb:blue,slider-swells-flow:blue,slider-swells-eq:amber"
></div>

1. Switch to **Asterion** with the same fader positions
2. Asterion is spectral -- it decomposes the sound into otherworldly fragments
3. Sweep **EBB** -- Asterion's EBB creates different spectral shifts than Velvet's
4. Sweep **EQ** -- tonal changes are more dramatic in spectral algorithms
5. Try feeding different source material: a chord, a voice, a drum hit. Notice how Asterion transforms each differently

### Exercise 3: Deadspace -- The Void (5 min)

<div data-swells-panel
  data-highlights="slider-swells-decay:amber,slider-swells-hi-damp:amber,slider-swells-ebb:blue,slider-swells-flow:blue"
></div>

1. Switch to **Deadspace**. This is the darkest, most void-like algorithm
2. Increase **DECAY** to maximum -- the sound disappears into a void that never ends
3. Reduce **HI DAMP** to minimum -- this makes the void even darker
4. Sweep **EBB** and **FLOW** -- Deadspace uses these for hollow, cavernous modulation
5. This algorithm excels at horror-adjacent and dark ambient textures

### Exercise 4: Build an Ethereal Texture (3 min)

<div data-swells-panel
  data-highlights="slider-swells-decay:amber,slider-swells-size:amber,slider-swells-ebb:blue,slider-swells-flow:blue,slider-swells-mix:amber"
></div>

1. Select **Velvet** (or whichever ethereal algorithm you preferred)
2. Set DECAY to **3 o'clock**, SIZE to **3 o'clock**, MIX to **fully wet**
3. Set EBB and FLOW to positions that create the most interesting texture
4. Play a single sustained note or chord and let the texture build
5. This is your ethereal texture patch -- document the settings

## What You Learned

- Velvet produces lush, harmonic shimmer
- Asterion creates spectral, otherworldly decomposition
- Deadspace generates void-like, dark ambient textures
- These algorithms benefit from longer DECAY and wet-biased MIX

## Document Your Results

Record your ethereal texture patch settings: algorithm choice, all fader positions, and what source material you used. Note which algorithm surprised you most.
