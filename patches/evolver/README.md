# Evolver Patch Library

Documented patches created during learning sessions. Each patch file includes full parameter values so you can recreate it from scratch.

## Patch Template

When saving a patch, create a file here with this format:

```markdown
# [Patch Name]

**Session**: XX | **Location**: P:XXX B:X | **Type**: bass/lead/pad/texture/drum/sequence

## Description
One sentence: what this sounds like and when you'd use it.

## Key Parameters
(Only the values that differ from the basic patch)

| Parameter | Value |
|-----------|-------|
| ... | ... |

## Playing Tips
How to play this patch expressively (mod wheel, aftertouch, velocity).

## Tags
#evolver #patch #[type]
```

## Naming Convention

- `basic-patch.md` — the reference patch
- `exercise-XX-description.md` — patches from session exercises
- `song-description.md` — patches used in actual compositions
