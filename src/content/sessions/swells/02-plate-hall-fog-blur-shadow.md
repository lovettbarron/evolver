# Session 02: Traditional Spaces -- Fog, Blur & Shadow

**Objective:** Compare the three traditional-space algorithms (Fog, Blur, Shadow) and understand how EBB and FLOW behave differently in each.

> [!tip] If you only have 5 minutes
> Select Fog, set DECAY to 2 o'clock. Listen. Switch to Blur -- same settings, different character. Switch to Shadow -- darker, deeper. The algorithms are the personality; the faders are the shape.

## Warm-Up (2 min)

Patch audio to IN L and L output to your mixer. Set the init state. Select Fog. Sweep DECAY and SIZE to remind yourself of the basic controls from Session 01.

## Setup

1. Init state (see [Swells Overview](/modules/swells/))
2. Audio source to **IN L**, **L** output to mixer
3. DECAY at **2 o'clock**, SIZE at **1 o'clock**, MIX at **12 o'clock**
4. Start with **Fog** algorithm

## Exercises

### Exercise 1: Fog -- Diffuse and Cloudy (5 min)

<div data-swells-panel
  data-highlights="slider-swells-decay:amber,slider-swells-size:amber,slider-swells-ebb:blue,slider-swells-flow:blue"
></div>

1. Play audio through Fog with the setup above. Listen to its diffuse, cloudy character
2. Sweep **EBB** from minimum to maximum -- note what it does in Fog mode
3. Sweep **FLOW** from minimum to maximum -- note the difference from EBB
4. Find a Fog setting you like. Note the fader positions

### Exercise 2: Blur -- Smeared and Soft (5 min)

<div data-swells-panel
  data-highlights="slider-swells-ebb:blue,slider-swells-flow:blue"
></div>

1. Switch to **Blur** (use MODEL UP/DOWN). Keep all other faders the same
2. Listen to how the reverb character changes -- Blur smears the transients more than Fog
3. Sweep **EBB** in Blur mode -- does it feel different from Fog's EBB?
4. Sweep **FLOW** in Blur mode -- compare to Fog
5. Find a Blur setting you like. Note the fader positions

### Exercise 3: Shadow -- Dark and Cavernous (5 min)

<div data-swells-panel
  data-highlights="slider-swells-hi-damp:amber,slider-swells-lo-damp:amber,slider-swells-ebb:blue,slider-swells-flow:blue"
></div>

1. Switch to **Shadow**. Keep all other faders the same
2. Shadow is inherently darker -- notice the low-end emphasis
3. Sweep **HI DAMP** -- Shadow responds dramatically to high-frequency damping
4. Sweep **EBB** and **FLOW** in Shadow mode -- these create cavernous, evolving textures
5. Find a Shadow setting you like. Note the fader positions

### Exercise 4: A/B/C Comparison (3 min)

1. Set all faders to the same position (your favorite from the three)
2. Switch between Fog, Blur, and Shadow rapidly using MODEL buttons
3. Listen for the personality differences with identical parameter settings
4. Which algorithm suits which type of source material best?

## What You Learned

- Fog produces diffuse, cloud-like reverb
- Blur smears transients and softens the sound
- Shadow emphasizes low frequencies and creates dark spaces
- EBB and FLOW have different effects in each algorithm

## Document Your Results

Save your favorite fader positions for each of the three algorithms. Note which source material (percussive, sustained, melodic) worked best with each algorithm.
