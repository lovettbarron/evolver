# Mixer

## Purpose

Combines oscillator outputs, noise, sub-oscillator, and external input into a single signal.

## Controls

| Control | Type | Range | Notes |
|---------|------|-------|-------|
| VCO A | Knob | 0-100% | Main oscillator level |

## Patch Points

| Jack | Type | Normalled To | Notes |
|------|------|-------------|-------|
| IN | Input | VCO A Out | Main oscillator input |
