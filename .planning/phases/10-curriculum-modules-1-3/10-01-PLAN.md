**All 3 sessions MUST follow this structure (per D-16, D-17, D-19, D-20):**
- YAML frontmatter: title, module: "Foundations", session_number (1/2/3), duration (20 or 25), prerequisite (null for 1, previous number for 2-3), output_type (technique or patch), difficulty: beginner, tags array, instrument: cascadia, reference citing Cascadia Manual pages

- H2 Setup with specific instructions and percentage values for knobs (per D-09)
- Cable summary table for sessions that use cables (per D-08): `> **Cables for this session:** | # | From | To | Purpose | Overrides |`
- H2 Exercises with numbered steps, each step having specific percentage values AND expected audible results ("you should hear...")
- `> [!info] Normalled:` callouts the FIRST time each normalled connection is used (per D-19 / CCURR-04), stating what patching overrides
- `> [!info] Cascadia's...` callouts for unique features (per D-17 / CCURR-03)
- H2 Exploration (optional, hyperfocus days) with 2-3 bounded suggestions
- H2 Output Checklist with checkboxes including "Session logged in Obsidian daily note"
- H2 Key Takeaways (2-3 bullets)
- H2 Next Session Preview

**Session 1: Orientation and First Sound (~20 min) (per D-02, D-07 / CCURR-05)**
- Concept section: "What Is a Semi-Modular Synthesizer?" -- explain normalling, signal path, modules as building blocks
- Exercises: Connect MIDI + audio, power on, play notes to hear normalled default (VCO A saw -> mixer -> VCF -> wave folder -> VCA A -> output), identify the sound character
- Normalling callouts needed: MIDI/CV -> VCO A Pitch, MIDI/CV -> VCO B Pitch, VCO A -> Mixer, Mixer OUT -> VCF IN, VCF OUT -> Wave Folder IN, Wave Folder -> VCA A, VCA A -> Output Control MAIN 1, Envelope A -> VCA A LEVEL MOD, MIDI/CV -> Envelope A Gate, MIDI/CV -> Envelope B Gate
- Use exact control names from module docs (e.g., "OCTAVE" not "octave selector", "PITCH" not "tuning knob")
- output_type: technique
- Cascadia-unique: normalling philosophy ("plays with zero cables but rewards patching")

**Session 2: Pulse Width Modulation and Sub-Oscillator (~25 min) (per D-02 / CCURR-05)**
- Concept section: "What Is Pulse Width Modulation?" and "What Is a Sub-Oscillator?" (can be two short paragraphs)
- Exercises: Raise PW MOD slider to hear LFO Y modulating pulse width (normalled), adjust PW slider for base pulse width (~25%, ~50%, ~75%), raise SUB slider in Mixer (~50%), explore SUB TYPE switch (-1 oct, OR mode, -2 oct), combine PWM + sub for thick sound
- Normalling callouts: LFO Y -> VCO A PWM IN
- output_type: technique
- Cascadia-unique: LFO Y pre-normalled to PWM (no cable needed), three sub-oscillator types including OR mode

**Session 3: Filter Envelope, Wave Folding, FM and FX (~25 min) (per D-02 / CCURR-05)**
- Concept section: "How Does a Filter Envelope Shape Timbre?" (covering filter envelopes, wave folding overview, FM overview -- brief since the session covers multiple topics as a "taste")
- Exercises: Adjust Envelope B RISE/FALL to hear filter sweep (normalled to VCF FM 1), adjust VCF FREQ (~25% to ~75%) and Q (~30% to ~60%), raise Wave Folder FOLD slider (~25% to ~75%) to hear harmonics, raise VCO A INDEX slider (~30%) to taste FM (VCO B sine normalled to FM 2), brief FX send/return overview (connect pedal if available)
- Keep FX to 2-3 minutes -- a taste, not deep dive (per research pitfall #1)
- Normalling callouts: Envelope B -> VCF FM 1, VCO B Sine -> VCO A FM 2, MIDI/CV -> VCF FM 2 (keyboard tracking)
- output_type: patch -- this session produces the "foundations-filter-sweep" curriculum patch
- Output checklist must include: saved patch documented in patches/cascadia/
- Cascadia-unique: Wave folder position after VCF in signal chain (unusual), FX send/return anywhere in chain

**Anti-patterns to avoid:**
- NO clock positions in session text (use percentages per D-09/D-10)
- NO vague knob instructions ("turn up the filter" -- always "Set VCF FREQ to ~75%")
- NO exercise steps without audible results ("you should hear...")
- NO normalled connections used without a [!info] callout the first time
- NO cable instructions without stating what normalling they override
- NO session exceeding 30 minutes or having more than 4 exercises
