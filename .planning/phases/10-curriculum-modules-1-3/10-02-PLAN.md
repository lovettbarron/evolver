**Session 4: VCO A Shapes and Tuning (~20 min) (per D-03)**
- Concept section: "What Are Oscillator Waveshapes?" -- explain how waveshape determines timbre (fundamental + harmonics), describe saw, pulse/square, triangle, sine characters
- Exercises: Hear each VCO A output via Mixer -- Saw slider (normalled), Pulse slider, raise Mixer IN 2 for sine (normalled from VCO A sine). Adjust OCTAVE selector through range. Use PITCH knob for fine tuning. Compare waveshapes at different octaves. Explore PULSE POSITION switch (center vs edge).
- Normalling callouts: VCO A -> Mixer (saw/pulse direct), Mixer IN 2 normalled from VCO A Sine
- output_type: technique
- Cascadia-unique: Mixer direct outs (SAW OUT, PULSE OUT), IN 2 normalled from VCO A sine, PULSE POSITION switch

**Session 5: VCO B, FM Synthesis and Sync (~25 min) (per D-03)**
- Concept section: "What Is Frequency Modulation Synthesis?" -- explain FM as one oscillator modulating another's frequency, creating harmonic/inharmonic spectra. Brief mention of oscillator sync.
- Exercises: Raise INDEX slider to hear FM (VCO B sine normalled to FM 2 -- ~0% to ~60%), switch TZFM/EXP to compare, switch AC/DC coupling, detune VCO B with PITCH knob (set PITCH SOURCE to "B" mode), engage SYNC TYPE switch (hard then soft), adjust VCO A PITCH while sync is on to hear harmonic sweeps
- Normalling callouts: VCO B Sine -> VCO A FM 2 IN, VCO B Saw -> VCO A SYNC IN
- output_type: patch -- produces the "fm-bell" curriculum patch
- Output checklist must include: saved FM bell patch
- Cascadia-unique: Through-zero FM (analog TZFM is rare), AC/DC coupling switch, soft sync "flip" mode, VCO B's VCO/LFO switch

**Session 6: Wave Folder as Sound Shaper (~20 min) (per D-03, D-05)**
- Concept section: "What Is Wave Folding?" -- explain wave folding vs filtering vs distortion, West Coast vs East Coast synthesis approaches
- Exercises: Start with FOLD at minimum (~0%), gradually increase to ~25%, ~50%, ~75% hearing harmonics added. Patch a cable: VCO A Pulse Out -> Wave Folder IN (this overrides VCF -> Wave Folder normalling -- document this override). Modulate FOLD with an LFO: patch LFO X Out -> Wave Folder FOLD MOD IN. Compare wave-folded sound before vs after filter by re-patching.
- Cable summary table required (first session with intentional cables):
  - Cable 1: VCO A Pulse Out -> Wave Folder IN, overrides VCF -> Wave Folder normal
  - Cable 2: LFO X Out -> Wave Folder FOLD MOD IN, overrides nothing (FOLD MOD has no normal)
- Normalling callouts: VCF OUT -> Wave Folder IN, Wave Folder -> VCA A
- output_type: technique
- difficulty: intermediate (first cable patching session)
- Cascadia-unique: Wave folder position post-VCF (unusual -- most West Coast synths have it pre-filter), FOLD MOD input for CV control

**Anti-patterns to avoid:**
- NO clock positions in session text (percentages only per D-09)
- NO vague knob instructions
- NO exercise steps without audible results
- NO normalled connections used without [!info] callout first time
- NO cable instructions without override context
- NO sessions over 30 minutes or more than 4 exercises
