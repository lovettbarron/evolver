- Set `"jsx": "preserve"`
- Set `"module": "esnext"` (Next.js requires this, not NodeNext)
- Set `"moduleResolution": "bundler"`
- Add `"plugins": [{ "name": "next" }]`
- Add `"next"` to types array: `"types": ["next"]`
- Keep `"paths": { "@/*": ["./src/*"] }`
- Remove `"outDir"`, `"rootDir"`, `"declaration"`, `"declarationMap"` (Next.js manages build output)
- Keep `"strict": true`, `"esModuleInterop": true`, `"skipLibCheck": true`
- Add `"incremental": true`
- Include: `["src/**/*", "next-env.d.ts"]`
