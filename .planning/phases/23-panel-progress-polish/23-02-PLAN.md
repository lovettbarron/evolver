**Add getHeatmapData to src/lib/practice-metrics.ts:**

Export a new function with this exact signature and behavior:
```typescript
export function getHeatmapData(
  completionDates: string[],
  weeks: number = 12,
): Array<{ date: string; count: number; dayOfWeek: number; weekIndex: number }>
```

Implementation:
1. Compute `today = new Date()` and `startDate` = today minus `weeks * 7 - 1` days
2. Align startDate backward to Monday: `const dayOffset = (startDate.getDay() + 6) % 7; startDate.setDate(startDate.getDate() - dayOffset);`
3. Build a `Map<string, number>` from completionDates (count occurrences per ISO date string)
4. Iterate from startDate to today, producing one cell per day:
   - `date`: ISO string YYYY-MM-DD
   - `count`: from the map, or 0
   - `dayOfWeek`: 0=Monday through 6=Sunday (using `(cursor.getDay() + 6) % 7`)
   - `weekIndex`: incrementing week counter
5. Return the cells array
