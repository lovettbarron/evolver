    **B. Create practice metrics utilities** (`src/lib/practice-metrics.ts`):

    Three pure exported functions:

    1. `getSessionsThisMonth(completionDates: Date[], currentDate?: Date): number`
       - Default `currentDate` to `new Date()`
       - Filter dates where `getMonth() === currentDate.getMonth() && getFullYear() === currentDate.getFullYear()`
       - Return count

    2. `getActiveWeeks(completionDates: Date[]): number`
       - For each date, compute ISO week key: `${year}-W${weekNumber}`
       - Use `Set<string>` to count distinct weeks
       - Return `weeks.size`
       - Helper: `getISOWeekKey(date: Date): string` — compute ISO 8601 week number (Monday-start). Simple algorithm: get Thursday of the date's week, compute days since Jan 1 of that Thursday's year, divide by 7, floor + 1.

    3. `getSyntheticCompletionDates(instrument?: string): Map<number, string>`
       - Import `SYNTHETIC_JOURNEY_WEEKS` and `SYNTHETIC_CASCADIA_JOURNEY_WEEKS` from `./synthetic-daily-notes`
       - Evolver reference start: `2026-02-01` (Sunday of week 1)
       - Cascadia reference start: `2026-02-22`
       - For each week in the journey, for each session in that week, compute a date: `referenceStart + (week - 1) * 7 days + sessionIndexInWeek days`
       - Return `Map<number, string>` where key is session number, value is ISO date string (YYYY-MM-DD)
