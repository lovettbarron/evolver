  <action>
1. Create src/lib/markdown/plugins/mermaid-placeholder.ts:
   - Custom rehype plugin that finds `<pre><code class="language-mermaid">` nodes
   - Replaces them with `<div class="mermaid-placeholder" data-chart="{content}"><noscript>Mermaid diagram (requires JavaScript)</noscript></div>`
   - MUST run BEFORE rehype-highlight to prevent mermaid code from being syntax-highlighted
   - Export as rehypeMermaidPlaceholder

2. Create src/lib/markdown/plugins/param-table.ts:
   - Custom rehype plugin that finds `<table>` elements
   - Checks if first `<th>` text content matches one of: "Parameter", "Param", "Name", "Control"
   - If match, wraps the `<table>` in `<div class="param-table">`
   - Adds class "param-name" to first `<td>` and "param-value" to second `<td>` in each row
   - Export as rehypeParamTable

3. Create src/lib/markdown/plugins/obsidian-tags.ts:
   - Custom rehype plugin that processes text nodes
   - Detects pattern: whitespace or start-of-text followed by `#` then `[a-zA-Z0-9_-]+`
   - Must NOT match line-start `#` (those are headings, already parsed)
   - Wraps matches in `<span class="obsidian-tag" data-tag="{tag-name}">#tag-name</span>`
   - Export as rehypeObsidianTags

4. Create src/lib/markdown/plugins/obsidian-embeds.ts:
   - Custom remark plugin (runs on mdast, before remark-rehype)
   - Detects `![[filename]]` pattern in text nodes
   - Replaces with a raw HTML node: `<div class="obsidian-embed" data-embed="{filename}"><p class="embed-placeholder">Embedded content: {filename}</p></div>`
   - Export as remarkObsidianEmbeds

5. Create src/lib/markdown/processor.ts:
   - Import all remark/rehype plugins from RESEARCH.md standard stack
   - Import all 4 custom plugins
   - createMarkdownProcessor(wikilinkPermalinks: string[]): returns configured unified processor
   - Plugin order (CRITICAL -- matches RESEARCH.md + UI-SPEC.md):
     1. remarkParse
     2. remarkFrontmatter(['yaml']) -- strip YAML from output
     3. remarkGfm -- tables, task lists
     4. remarkObsidianEmbeds -- before wikilinks
     5. remarkWikiLink configured with: permalinks array, hrefTemplate: (permalink) => `/instruments/evolver/${permalink}`, aliasDivider: '|', wikiLinkClassName: 'wikilink'
     6. remarkRehype({ allowDangerousHtml: true })
     7. rehypeRaw
     8. rehypeMermaidPlaceholder -- BEFORE highlight
     9. rehypeCallouts
     10. rehypeHighlight
     11. rehypeParamTable
     12. rehypeObsidianTags
     13. rehypeSlug
     14. rehypeAutolinkHeadings({ behavior: 'prepend', properties: { className: ['heading-anchor'], ariaHidden: 'true' }, content: { type: 'text', value: '#' } })
     15. rehypeStringify
   - renderMarkdown(content: string, permalinks?: string[]): Promise<string> -- creates processor and processes content
   - Handle broken wikilinks: configure remark-wiki-link to add class "wikilink-broken" and href="#" for links not in permalinks array
   - Export createMarkdownProcessor and renderMarkdown
