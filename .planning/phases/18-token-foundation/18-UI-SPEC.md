## Spacing Scale

Declared values (multiples of 4 only). Existing scale retained per D-07.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, checkbox margin-right |
| sm | 8px | Compact element spacing, table cell padding |
| md | 16px | Default element spacing, callout/pre padding |
| lg | 24px | Section padding, mermaid placeholder padding |
| xl | 32px | Layout gaps, mermaid placeholder vertical margin |
| 2xl | 48px | Major section breaks, scroll-padding-top |
| 3xl | 64px | Page-level spacing |

Exceptions: none

**Migration note (D-08):** All 51 components migrate from inline Tailwind spacing classes (p-4, gap-6, etc.) to these token references (p-md, gap-lg, etc.) in this phase. The scale values do not change — only the indirection does.

## Color

### OKLCH Palette — Warm Dark Surfaces (D-01, D-04, D-05, D-06)

All surfaces use OKLCH hue ~85 (olive family). Lightness increases with elevation.

| Token | OKLCH Value | Role | 60/30/10 |
|-------|-------------|------|----------|
| `--color-bg` | oklch(0.12 0.01 85) | Page background, dominant surface | 60% — dominant |
| `--color-sunken` | oklch(0.10 0.01 85) | Recessed areas, footer background | 60% — dominant |
| `--color-surface` | oklch(0.16 0.01 85) | Card backgrounds, code blocks, table stripes | 30% — secondary |
| `--color-surface-raised` | oklch(0.20 0.015 85) | Elevated cards, hover states, overlay panels | 30% — secondary |
| `--color-overlay` | oklch(0.24 0.015 85) | Modal overlays, dropdown menus, tooltips | 30% — secondary |

### Semantic Color Tokens

| Token | OKLCH Value | Usage |
|-------|-------------|-------|
| `--color-text` | oklch(0.93 0.01 85) | Primary text — warm white, not pure #e8e8e8 |
| `--color-muted` | oklch(0.55 0.01 85) | Secondary text, table headers, timestamps (D-03: warmed AND lightened) |
| `--color-accent` | oklch(0.85 0.18 105) | Primary accent — warmed toward yellow-green per D-02 (hue 105-110 range) |
| `--color-param` | oklch(0.72 0.15 120) | Parameter values in code/tables — warm green |
| `--color-border` | oklch(0.25 0.01 85) | Primary borders — warm olive, not neutral gray (D-06) |
| `--color-border-subtle` | oklch(0.20 0.008 85) | Subtle dividers — near-invisible warm separation |
| `--color-destructive` | oklch(0.55 0.2 25) | Destructive actions only — warm red |

### 60/30/10 Distribution

- **60% dominant:** `--color-bg` and `--color-sunken` — warm dark olive surfaces that fill most of the viewport
- **30% secondary:** `--color-surface`, `--color-surface-raised`, `--color-overlay` — card and container backgrounds
- **10% accent:** `--color-accent` reserved for: link hover states, callout left borders, accent-color on checkboxes, pulse-glow animation, active navigation indicators. Never used for backgrounds or large surface areas.

### Contrast Requirements (D-10, TOKEN-04)

All pairings must meet WCAG AA (4.5:1 minimum). Critical pairings to validate:

| Foreground | Background | Minimum Ratio | Purpose |
|------------|------------|---------------|---------|
| `--color-text` | `--color-bg` | 4.5:1 | Body text on page background |
| `--color-text` | `--color-surface` | 4.5:1 | Body text on card backgrounds |
| `--color-text` | `--color-surface-raised` | 4.5:1 | Body text on elevated cards |
| `--color-muted` | `--color-bg` | 4.5:1 | Secondary text on page background |
| `--color-muted` | `--color-surface` | 4.5:1 | Table headers on card backgrounds |
| `--color-accent` | `--color-bg` | 4.5:1 | Accent text/links on page background |
| `--color-accent` | `--color-surface` | 4.5:1 | Accent text on card backgrounds |
| `--color-param` | `--color-surface` | 4.5:1 | Code/param text on code block backgrounds |

**Validation approach (D-10):** Automated OKLCH contrast script (no browser needed) AND a `/dev/tokens` visual page showing all pairings with pass/fail badges.

### Grain Texture (D-05)

- Pre-rendered 64x64 WebP tile at `public/textures/grain.webp`
- Applied via CSS pseudo-element at opacity 0.02-0.05
- GPU composited, zero JS cost
- Applied to `--color-bg` surface only (not cards or overlays)
