## Component Inventory

### Primary Component

**`EvolverPanel`** -- Single reusable React component rendering the Evolver panel as inline JSX SVG.

Props interface:

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `knobValues` | `Record<string, number>` | No | MIDI values (0-127) keyed by control ID. Drives indicator rotation. |
| `highlights` | `Array<{ controlId: string; color: 'blue' \| 'amber' }>` | No | Controls to highlight with glow rings for curriculum annotations. |
| `activeSections` | `string[]` | No | Section names to tint (e.g., "oscillators", "filter", "amp"). |
| `onKnobChange` | `(controlId: string, value: number) => void` | No | Callback when user drags a knob. Omit to make knobs display-only. |
| `className` | `string` | No | Additional CSS class for the container. |

### Integration Wrappers

| Component | Context | Behavior |
|-----------|---------|----------|
| `SessionPanelSidebar` | Session detail page | Collapsible sidebar, receives highlights and activeSections from session metadata |
| `PatchPanelView` | Patch detail page | Inline below patch description, receives knobValues from patch parameters |
| Standalone route | `/instruments/evolver/panel` | Full-width exploratory view, all knobs interactive, no annotations |
| Quick-ref integration | Quick-ref panel | Compact neutral view, display-only |

Source: D-07, D-08, D-09 from CONTEXT.md.

## Interaction Contract

### Knob Drag (D-10)

| Property | Value |
|----------|-------|
| Gesture | Vertical drag: drag up increases value, drag down decreases |
| Sensitivity | 1 MIDI unit per 3px of vertical movement |
| Range | 0-127 integer values |
| Visual feedback | Indicator line rotates from -135deg (value 0, 7 o'clock) to +135deg (value 127, 5 o'clock) |
| Drag active ring | `#c8ff00` at 40% opacity circle around knob during drag |
| Cursor | `grab` on hover, `grabbing` during drag |
| Touch support | Vertical touch drag with same sensitivity |
| onChange firing | Fires on every value change during drag (not just on release) |

### Tooltip (D-05, D-12)

| Property | Value |
|----------|-------|
| Trigger | Mouse hover on any control (knob, switch, LED) |
| Delay | 200ms enter, 0ms leave |
| Position | Centered above the control, 8px gap. Flip below if near top edge of SVG. |
| Size | Auto-width, max-width 200px |
| Background | `#161616` with 1px `#333333` border, 8px border-radius |
| Padding | 8px horizontal, 8px vertical |
| Content line 1 | Control name in 13px Inter bold (700), `#e8e8e8` |
| Content line 2 | Value (if set) in 13px JetBrains Mono regular (400), `#a3e635` |
| Content line 3 | NRPN number in 13px Inter regular (400), `#737373` |
| Z-index | Above SVG content, below app-level modals (z-30) |
| Pointer events | none (tooltip does not block interaction) |

### Glow Ring (D-04)

| Property | Value |
|----------|-------|
| Shape | Circle, radius = knob radius + 4px |
| Blue glow | `#3388ff`, opacity 0.6, SVG filter with 3px gaussian blur |
| Amber glow | `#ffaa33`, opacity 0.6, SVG filter with 3px gaussian blur |
| Animation | Fade in 200ms ease-out on mount, no pulse/blink |
| Stacking | Glow renders behind the knob circle, in front of panel background |

### Section Tint (D-06)

| Property | Value |
|----------|-------|
| Shape | Rectangle covering the section bounding box with 4px border-radius |
| Fill | `#3388ff` at 8% opacity |
| Animation | Fade in 150ms ease-out |
| Sections available | Envelope 3, LFO, Oscillators, External Input, Mixer, Filter, Envelopes 1+2, VCA, Output, Feedback, Delay, Sequencer, Modulators, Noise, Distortion |

### Collapsible Sidebar (D-08)

| Property | Value |
|----------|-------|
| Width | 400px fixed, collapsible to 0px |
| Position | Right side of session content area |
| Toggle | Button with "Show Panel" / "Hide Panel" text, accent color |
| Animation | Width transition 200ms ease-out |
| Breakpoint | Hidden below 1024px viewport width (lg breakpoint). Session content uses full width. |
| Default state | Collapsed on first visit |
| Persistence | No persistence -- resets to collapsed on navigation |

### Switches and LEDs (D-11)

| Property | Value |
|----------|-------|
| Interaction | Display-only, not clickable or toggleable |
| Cursor | `default` (no pointer) |
| Tooltip | Yes -- shows name and NRPN on hover, same as knobs |

## Layout Contract

Primary visual anchor: the EvolverPanel SVG. All layout contexts orient around the panel as the focal point of this phase.

### Session Detail with Panel Sidebar

```
+--------------------------------------------------+
| Sticky Header                                     |
+-----------------------------------+---------------+
| Session Content (max-w-720px)     | Panel Sidebar |
| - prose markdown                  | (400px, right)|
| - exercises                       | - EvolverPanel|
|                                   | - collapsible |
+-----------------------------------+---------------+
```

- Below `lg` (1024px): sidebar hidden, full-width session content (existing behavior preserved)
- At `lg` and above: sidebar visible as collapsible panel to the right
- Session content `max-w-720px` constraint relaxes to `max-w-[calc(100%-400px-32px)]` when sidebar is open

### Patch Detail with Panel

```
+--------------------------------------------------+
| Sticky Header                                     |
+--------------------------------------------------+
| Patch Info (name, type, origin)                   |
| Audio preview placeholder                         |
| Prose content                                     |
| Cable routing (if Cascadia)                       |
| Knob settings table                               |
| EvolverPanel (inline, full content width)          |
+--------------------------------------------------+
```

- Panel renders inline below existing patch content, above prev/next nav
- Only renders for Evolver patches (instrument check)
- Full content width within the `max-w-720px` container

### Standalone Route (`/instruments/evolver/panel`)

```
+--------------------------------------------------+
| Sticky Header (back to Evolver overview)          |
+--------------------------------------------------+
| Page title: "Evolver Panel"                       |
| EvolverPanel (full width, max-w-1200px, centered) |
| All knobs interactive, no annotations              |
+--------------------------------------------------+
```

## SVG Conversion Notes

Source: `references/evolver-panel.svg` (1200x520 viewBox)

| Property | Value |
|----------|-------|
| Approach | Inline JSX -- convert SVG elements to React JSX with camelCase attributes |
| CSS classes | Convert to Tailwind utilities or CSS modules scoped to component |
| Control IDs | Preserve all existing `id` attributes (e.g., `knob-env3-destination`) as React refs/data attributes |
| Indicator rotation | Apply via `transform: rotate({deg}deg)` on the `<line>` element within each knob `<g>` |
| SVG filters | Add `<filter id="glow-blue">` and `<filter id="glow-amber">` to `<defs>` for annotation glows |
| Responsive | SVG scales via `width="100%" viewBox="0 0 1200 520"` preserving aspect ratio |
| Feedback section fix (D-03) | Frequency on top, Level as large knob below, Grunge as switch not knob -- fix before JSX conversion |
