Create `src/components/evolver-panel-tooltip.tsx` as an HTML overlay positioned via `getBoundingClientRect`. This is NOT an SVG foreignObject (per RESEARCH.md anti-pattern).

**Props interface:**
```typescript
interface PanelTooltipProps {
  controlId: string | null;
  svgRef: React.RefObject<SVGSVGElement | null>;
  knobValues?: Record<string, number>;
  highlightTarget?: number;  // target value from session context
}
```

**Positioning logic:**
- When `controlId` changes to non-null, find the SVG element via `svgRef.current.querySelector(`#${controlId}`)`
- Call `getBoundingClientRect()` on the element to get screen-space coordinates
- Position tooltip centered above the element: `left = rect.left + rect.width / 2`, `top = rect.top - 8`
- Transform: `translate(-50%, -100%)` to center horizontally and place above
- If `top` would place tooltip above viewport (< 0), flip below: `top = rect.bottom + 8`, transform removes the -100%

**Tooltip content (per UI-SPEC Copywriting Contract):**
- Line 1: Control name — 13px Inter bold (#e8e8e8 / text-text)
- Line 2: Value (if knobValues[controlId] exists) — 13px JetBrains Mono regular (#a3e635 / param color), format: `"{value} / 127"`
- Line 2 alt: Target value (if highlightTarget provided in session context) — format: `"Set to {target}"`
- Line 3: NRPN number (if not null) — 13px Inter regular (#737373), format: `"NRPN {number}"`

**Visual styling (exact values from UI-SPEC):**
- Background: `#161616` (bg-surface)
- Border: 1px solid `#333333`
- Border radius: 8px
- Padding: 8px horizontal, 8px vertical
- Max width: 200px
- Z-index: 30 (z-30)
- Pointer events: none
- Position: fixed

**Behavior:**
- Component is always rendered but returns null when controlId is null
- No enter delay for now (200ms delay deferred — start with instant show)
- Disappears instantly on mouse leave (0ms)
