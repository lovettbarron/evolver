Required frontmatter fields:
```yaml
name: "Patch Name"
type: bass|lead|pad|drum|texture|sequence|fx
session_origin: null       # All standalone per D-13
description: "One-line description"
tags: [tag1, tag2, cascadia]
instrument: cascadia
created: "2026-03-31"
audio_preview: "filename.mp3"  # Per D-09: files NOT shipped, just referenced
cable_routing:               # Per D-01, D-02, D-03: flat array, human labels
  - source: "Module Jack Out"
    destination: "Module Jack In"
    purpose: "What this cable does"
knob_settings:               # Per D-05, D-06, D-07: key controls that differ from default
  Module Name:               # Must match canonical module doc title exactly
    - control: "Control Name"
      value: "clock position or switch value"
```

Canonical module names (from module doc titles):
"VCO A", "VCO B", "VCF", "Wave Folder", "VCA A", "VCA B / LPF",
"Mixer", "Envelope A", "Envelope B", "LFO X/Y/Z", "Output Control",
"FX Send/Return", "MIDI/CV", "Patchbay", "Utilities", "Push Gate", "Line In"

Value format per D-06:
- Continuous knobs: clock positions ("2 o'clock", "10 o'clock", "full CW", "full CCW", "12 o'clock")
- Switches/toggles: actual values ("LP4", "ON", "Mid", "Saw")
- Selectors: named positions ("4", "6" for octave selectors)
