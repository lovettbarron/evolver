**Recipe session structure (per D-05, D-09):**
These differ from technique sessions. Each session IS a recipe -- the exercises walk through building the patch cable by cable, knob by knob. The structure is:
- YAML frontmatter: title, module: "Sound Design", session_number (20-25), duration (25 or 30), prerequisite (previous number), output_type: patch, difficulty: advanced, tags array with "curriculum" and "recipe", instrument: cascadia, reference citing relevant manual pages
- H1 title: "Session XX: [Category] Sound Design"
- **Objective:** "Build a [category] patch from scratch, documenting every cable and knob setting."
- `> [!tip] If you only have 5 minutes` -- describe the key character of the sound and 1-2 most important cables
- H2 concept section: "Sound Design Strategy: [Category]" -- explain what makes a great [bass/lead/pad/etc] patch in synthesis generally (1-2 paragraphs)
- H2 Warm-Up (2 min): return to normalled default, recall one technique from a previous module
- H2 "Building the Patch" (instead of generic "Exercises"):
  - Step-by-step numbered instructions, each step: (1) set a knob or patch a cable, (2) state what it does, (3) state what you should hear change
  - Cable summary table at the top: `| # | From | To | Purpose | Overrides |`
  - After the complete cable table, walk through knob settings module by module
- H2 "Playing Tips" -- how to play the patch expressively (velocity, different registers, held notes vs staccato)
- H2 "Variations" -- 2-3 quick modifications to explore
- H2 Output Checklist: must include "Patch documented in patches/cascadia/ with full cable routing and knob settings" and "Session logged in Obsidian daily note"
- H2 Key Takeaways (2-3 bullets about what synthesis techniques make this patch work)
- H2 Next Session Preview (except Session 25 which says "Congratulations! You have completed the Cascadia curriculum.")

**CRITICAL: Patch type mapping for PatchSchema compatibility:**
The PatchSchema `type` enum allows: bass, lead, pad, drum, texture, sequence, fx.
- Session 20 (Bass) -> patch type: "bass"
- Session 21 (Lead) -> patch type: "lead"
- Session 22 (Pad) -> patch type: "pad"
- Session 23 (Percussion) -> patch type: "drum" (NOT "percussion" -- schema uses "drum")
- Session 24 (Texture) -> patch type: "texture"
- Session 25 (Ambient) -> patch type: "pad" (closest match -- ambient is not in schema, use "pad" since ambient patches are slow/evolving like pads)

**Session 20: Bass Sound Design (~25 min)**
- Build a deep, usable bass patch. Low octave setting on VCO A, saw + sub via Mixer, VCF in LP4 mode with low FREQ and moderate Q, Envelope A for tight amplitude envelope, Envelope B for subtle filter movement. Zero or minimal cables (mostly normalled path).
- Patch name: "Deep Sub Bass Recipe"
- Cable count: 0-1

**Session 21: Lead Sound Design (~25 min)**
- Build a cutting, expressive lead. VCO A saw or pulse, add VCO B for slight detune or FM, VCF in LP2 or BP2 mode for presence, moderate resonance, Envelope B for filter bite on attack. Wave folder for edge.
- Patch name: "Searing Lead Recipe"
- Cable count: 1-2

**Session 22: Pad Sound Design (~25 min)**
- Build an evolving, textured pad. Slow LFO modulation on filter and/or PWM, long Envelope A attack/release for swells, VCF in LP4 mode with moderate FREQ, Wave Folder at low amount for subtle harmonics, possibly S&H -> Slew for gentle pitch drift.
- Patch name: "Evolving Pad Recipe"
- Cable count: 2-3

**Session 23: Percussion Sound Design (~30 min)**
- Build complex percussion using LPG (from Module 4) plus additional modulation. Short Envelope A, VCA B/LPF in LPG mode, layer noise from Mixer, try different source waveforms, add ring mod character. Produce a more complex version of the Module 3 LPG bongo.
- Patch name: "Complex Percussion Recipe"
- Cable count: 3-4

**Session 24: Texture Sound Design (~25 min)**
- Build an evolving noise/texture. Noise through VCF with S&H modulating cutoff, Wave Folder for harmonic content, LFO-modulated parameters for movement, possibly self-patching for complexity.
- Patch name: "Granular Texture Recipe"
- Cable count: 3-5

**Session 25: Ambient Sound Design (~30 min)**
- Build a spacious, evolving ambient patch. Combines slow LFO modulation, S&H -> Slew for drift, long envelopes, wave folding, FX send/return (if pedal available, or describe the intended FX routing). This is the capstone -- uses techniques from every prior module.
- Patch name: "Ambient Drift Recipe"
- Cable count: 4-6
- End with congratulatory outro: "You have completed the 25-session Cascadia curriculum."

**Anti-patterns:**
- NO clock positions in session text (percentages only per D-07)
- NO vague instructions -- every cable and knob must be specific
- NO "turn the filter knob" -- always "Set VCF FREQ to ~30%"
- Use EXACT control names from module docs
- Each session must be completable in 30 min or less
