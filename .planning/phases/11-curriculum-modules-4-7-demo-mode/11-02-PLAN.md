**Session 16: FM Chains and Cross-Modulation (~25 min)**
- Concept section: "What Is Cross-Modulation?" -- explain using one oscillator to modulate another at audio rates, sidebands, harmonic vs inharmonic spectra, FM vs ring modulation differences
- Exercises: (1) Patch VCO B -> VCO A FM 2 for basic FM (1 cable), explore INDEX amount for depth, (2) use Ring Mod output as a modulation source -- patch RING MOD OUT to VCF FM 3 (1 cable), hear how ring-modulated signal creates metallic filter sweeps, (3) create an FM chain: VCO B modulates VCO A, VCO A output through VCF with envelope modulation -- complete timbral sculpting
- Cable count: 2-3 (VCO B -> VCO A FM 2, RING MOD OUT -> VCF FM 3)
- Cascadia-unique: Ring Mod as both sound source AND modulation source, VCO B sine normalled to VCO A FM 2 (cable overrides normalling for explicit routing), 3 VCF FM inputs with different response curves
- output_type: technique
- session_number: 16, prerequisite: 15, difficulty: advanced

**Session 17: Self-Patching and Feedback Loops (~25 min)**
- Concept section: "What Is Self-Patching?" -- explain routing outputs back to inputs within the same instrument for complex timbres, controlled chaos, and emergent behavior. Explain why feedback is dangerous at high levels (protect your ears and speakers)
- Exercises: (1) Simple feedback: patch VCF MODE OUT -> Mixer external input or back to VCF LEVEL (1 cable) at LOW levels to create resonant character, (2) Wave folder feedback: patch Wave Folder OUT -> Mixer, then Mixer OUT back through VCF for complex harmonic stacking, (3) Envelope-controlled feedback: use Envelope A to modulate feedback amount via Mixuverter (attenuate the feedback path)
- Cable count: 2-4 (varies per exercise, clearly document each)
- **SAFETY WARNING:** Include a prominent `> [!warning]` callout about starting with ALL feedback amounts at MINIMUM and increasing slowly. Feedback can produce dangerously loud signals.
- Cascadia-unique: Multiple simultaneous outputs (LP4, HP4, MODE) allow creative feedback routing, Mixuverter as feedback attenuator
- output_type: technique
- session_number: 17, prerequisite: 16, difficulty: advanced

**Session 18: FX Pedal Integration (~20 min)**
- Concept section: "Why Use External Effects with a Synthesizer?" -- explain how reverb, delay, distortion, and other effects expand the sonic palette, send/return vs inline routing
- Exercises: (1) Connect an FX pedal to FX SEND OUT and FX RETURN IN (2 cables to pedal), set LINE/PEDAL switch to PEDAL, adjust DRY/WET mix, (2) explore placing FX at different points in the signal chain by patching the send from different module outputs, (3) use the phase correction on FX Return if needed
- Cable count: 2 external (to/from pedal) + 0-1 internal
- Note: If no pedal available, explain what WOULD happen and suggest using the FX loop as a signal routing tool (send to external mixer/recorder)
- Cascadia-unique: Built-in FX send/return with LINE/PEDAL impedance switch, DRY/WET mix, phase correction -- most semi-modulars lack this
- output_type: technique
- session_number: 18, prerequisite: 17, difficulty: intermediate (not advanced -- FX routing is conceptually simpler)

**Session 19: Audio-Rate Modulation Mastery (~25 min)**
- Concept section: "What Happens When Modulation Reaches Audio Rate?" -- explain the transition from LFO wobble to sideband generation, why audio-rate modulation creates new frequencies (sum and difference tones), the continuum from vibrato to FM
- Exercises: (1) Start with LFO X modulating VCO A pitch at slow rate (vibrato), gradually increase LFO rate into audio range -- hear the transition from wobble to new tones, (2) use VCO B as an LFO (set to lowest octave) modulating VCF cutoff via FM 3 for throbbing/growling effects, (3) stack audio-rate modulation: VCO B -> VCO A FM (pitch) while LFO at audio rate -> VCF FM (filter) for complex evolving textures, (4) use Ring Mod for classic AM/ring mod tones -- explore tuning relationships between VCO A and VCO B
- Cable count: 2-3 (LFO -> VCO A pitch, VCO B -> VCF FM 3, various)
- Cascadia-unique: LFO X/Y/Z reach audio rates (not all semi-modular LFOs do), Ring Mod always available as dedicated module
- output_type: technique
- session_number: 19, prerequisite: 18, difficulty: advanced
