- Exercises: Sweep through all 8 VCF modes (LP1/LP2/LP4/BP2/BP4/HP4/NT2/PHZR) using the MODE selector while playing notes, explore resonance (Q) from subtle to self-oscillation, demonstrate filter as sine oscillator at high Q
- Cable count: 0 (all normalled for basic filtering)
- Cascadia-unique: 8 filter modes (most semi-modulars have 1-2), simultaneous LP4/HP4/MODE outputs, LEVEL pre-gain for distortion character

**Session 11: Filter FM and Filter as Sound Source (~25 min)**
- Concept section: "What Is Filter Frequency Modulation?" -- explain using envelopes, LFOs, and audio-rate signals to modulate filter cutoff
- Exercises: Use Envelope B -> VCF FM 1 (normalled) with different envelope shapes, patch LFO to VCF FM 3 for rhythmic filter sweeps (1 cable), drive Q to self-oscillation and play the filter as a sine tone via keyboard tracking (normalled FM 2), patch VCO B -> VCF FM 3 for audio-rate filter FM
- Cable count: 2 (LFO -> FM 3, VCO B -> FM 3 in separate exercises)
- Cascadia-unique: 3 separate FM inputs with different scaling (envelope, 1V/oct, linear), FM 2 keyboard tracking for playable filter
- output_type: technique
- session_number: 11, prerequisite: 10

**Session 12: LPG Deep Dive and Percussion Palette (~25 min)**
- Concept section: "The Low Pass Gate: Amplitude Meets Filtering" -- explain how LPG couples volume and brightness decay, why it sounds organic, Buchla/west coast history (brief, building on Session 9 intro)
- Exercises: Build on Session 9's LPG bongo by exploring different source waveforms through VCA B/LPF (saw vs triangle vs noise), try different envelope shapes (Envelope A fast percussive, then slower for different timbres), combine LPG output with main signal path via Mixer, create a palette of 3-4 percussion tones
- Cable count: 2 (same as Session 9: VCO A -> VCA B IN, Envelope A -> VCA/LPF B CV IN)
- Cascadia-unique: VCA CONTROL switch (LPG mode vs LPF-only mode), VCA B normalled from Ring Mod OUT
- output_type: technique
- session_number: 12, prerequisite: 11

**Session 13: LFO Deep Dive (~25 min)**
- Concept section: "What Is a Low Frequency Oscillator?" -- explain LFOs as sub-audio modulation sources, rate, shape, destinations
- Exercises: Explore LFO X rate range (slow sweep to audio rate), use LFO Y -> VCO A PWM (normalled), patch LFO X to VCF FREQ for filter wobble, explore rate dividers (div3/div4/div5/div8) for polyrhythmic modulation, try LFO at audio rate for FM-like effects
- Cable count: 1-2 (LFO X -> VCF, LFO Z -> other destinations)
- Cascadia-unique: 3 linked triangle LFOs (X/Y/Z), rate dividers creating polyrhythmic relationships, LFO Y pre-normalled to PWM, LFO Z normalled to MULT IN 1
- output_type: technique
- session_number: 13, prerequisite: 12

**Session 14: Sample & Hold and Slew Limiter (~25 min)**
- Concept section: "What Are Sample & Hold and Slew Limiting?" -- explain S&H as voltage snapshots at clock rate (stepped random), slew as voltage smoothing (portamento, smoothed random)
- Exercises: Listen to S&H output driving VCF cutoff (stepped random), explore S&H clock sources (internal noise normalled, MIDI CLK), patch S&H -> Slew Limiter for smoothed random voltages, try different Slew directions (rise only, fall only, both) and shapes (linear, exponential), use slew-smoothed random -> VCO pitch for gentle pitch wandering
- Cable count: 2-3 (S&H OUT -> destinations, S&H -> Slew -> destinations)
- Cascadia-unique: S&H normalled from MIDI CLK + internal noise (works with zero cables), Slew Limiter 3-position direction switch, LIN/EXP shape switch
- output_type: technique
- session_number: 14, prerequisite: 13

**Session 15: Mixuverter and Voltage Processing (~20 min)**
- Concept section: "What Is an Attenuverter?" -- explain attenuation, inversion, offset, and why voltage processing is essential for modular patching
- Exercises: Use Mixuverter as a DC offset source (no input, turn knob to generate voltage), use as attenuverter on an LFO signal (reduce and invert modulation depth), combine two modulation sources (e.g., LFO + envelope) into one output, explore x2 doubling switch for signal boosting, use Mixuverter to create bipolar modulation from unipolar envelope
- Cable count: 2-3 (LFO -> Mixuverter IN, Mixuverter OUT -> destination)
- Cascadia-unique: Mixuverter doubles as DC source with no input, x2 switch for signal doubling, patchbay BI>UNI converter, MULT normalled from LFO Z
- output_type: technique
- session_number: 15, prerequisite: 14
