**Session 04: Crow Meets Just Friends** (intermediate, 30min)

First i2c contact. Cover:
1. Physical i2c connection: locate i2c headers on both modules, connect with a 3-pin cable (GND, SDA, SCL)
2. In Druid REPL, test connection: `ii.jf.trigger(1, 1)` -- should trigger channel 1 on JF (set JF to Shape/Transient mode)
3. Enter Synthesis mode: `ii.jf.mode(1)` -- JF enters a special i2c-only mode
4. Play a note: `ii.jf.play_note(0, 5)` -- play middle C at full level (0V = middle C, 5V level)
5. Trigger individual channels: `ii.jf.trigger(1, 1)` through `ii.jf.trigger(6, 1)` -- each channel fires independently
6. Exit Synthesis mode: `ii.jf.mode(0)` -- JF returns to panel control

Include `<div data-crow-panel data-highlights="usb-crow:blue">` and `<div data-just-friends-panel data-highlights="jack-jf-identity-out:amber,jack-jf-2n-out:amber">` markers where helpful.

Just Type commands used: `ii.jf.trigger(ch, state)`, `ii.jf.mode(state)`, `ii.jf.play_note(pitch, level)`

Output: documented first i2c connection + trigger/note test results

**Session 05: Polyphonic Sequencing & Voice Allocation** (advanced, 30min)

Build on session 04:
1. Use `ii.jf.play_note(pitch, level)` with sequins for a melodic sequence: `melody = sequins{0, 0.25, 0.5, 0.583}` (C, E, G, Bb in V/oct)
2. JF automatically allocates voices round-robin when using play_note
3. Use `ii.jf.play_voice(ch, pitch, level)` for specific voice control -- assign each channel a fixed note
4. Transpose all voices: `ii.jf.transpose(0.5)` -- shifts everything up a perfect fourth
5. Use `ii.jf.run_mode(1)` for sustaining voices (hold notes until released)
6. Build a chord player script using metro + sequins that cycles through chord voicings

Just Type commands used: `ii.jf.play_note`, `ii.jf.play_voice`, `ii.jf.transpose`, `ii.jf.run_mode`

Output: polyphonic sequencer script documented with code listing

**Session 06: Geode Rhythms & Generative Patches** (advanced, 30min)

The creative capstone:
1. Geode mode: `ii.jf.mode(1)` then use `ii.jf.tick(bpm)` to set internal clock -- JF generates polyrhythmic patterns across 6 channels based on INTONE setting
2. Experiment with different INTONE positions to change rhythmic ratios
3. Custom tuning: `ii.jf.retune(ch, num, den)` to set individual channel ratios (e.g., `ii.jf.retune(2, 3, 2)` for channel 2 at 3/2 ratio)
4. Build a generative patch: Crow input reads external CV, maps to JF tick rate. Crow ASL generates slowly evolving pitch sequences via play_note. JF INTONE knob adds rhythmic complexity
5. Combine Crow's scripting (input handlers, metro, sequins) with JF's six voices for a self-playing instrument

Just Type commands used: `ii.jf.tick`, `ii.jf.retune`, plus all previous commands

Output: generative patch script + recording of the self-playing system
