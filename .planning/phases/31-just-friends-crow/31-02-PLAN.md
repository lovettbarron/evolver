Session details per D-01:

1. **01 What is Crow** (beginner, 25min, section: Foundations): What Crow is (scriptable USB-CV bridge), physical setup (USB-C to computer, power from eurorack), installing Druid (`pip install monome-druid`), first REPL session. Exercise: type `output[1].volts = 3.0` in Druid and measure voltage with a multimeter or patch to another module. Explore `input[1].mode('stream', 0.1)` and see values print. Output: successful Druid connection + first voltage output documented.

2. **02 Input/Output Scripts** (beginner, 25min, section: Scripting): Read CV input and respond with output. Configure `input[1].mode('stream', 0.1)` with a handler that maps input voltage to output. Use `input[1].stream = function(v) output[1].volts = v * 2 end` for voltage scaling. Try `input[1].mode('change', 1.0, 0.1, 'rising')` for gate detection. Output: working input-to-output script documented with code listing.

3. **03 ASL Slopes** (intermediate, 30min, section: Scripting): A Slope Language for building envelopes and LFOs without external triggers. `output[1].action = loop{ to(5, 1), to(0, 1) }` creates a triangle LFO. Build an AD envelope: `output[1].action = once{ to(5, 0.01), to(0, 0.5) }`. Use `output[1]()` to trigger it. Explore `lfo()` helper and sequins for melodic sequences: `s = sequins{0, 2.5, 5, 7.5}; output[1].action = loop{ to(s(), 0.25) }`. Output: ASL envelope + LFO script documented.

Include `<div data-crow-panel>` markers where they aid understanding (highlighting active inputs/outputs).
