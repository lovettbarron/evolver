## Spacing Scale

Declared values (all multiples of 4, inherited from Phase 18 tokens):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, demo badge padding |
| sm | 8px | Compact element spacing, nav internal gaps |
| md | 16px | Default element spacing, footer link gaps |
| lg | 24px | Section padding, nav horizontal padding |
| xl | 32px | Layout gaps, footer vertical padding |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Nav bar height: 60px (within D-01 range of 56-64px; not on the 8-point scale but acceptable as a fixed layout dimension)
- Mobile hamburger touch target: 44px minimum (accessibility requirement)

Source: globals.css --spacing-* tokens (Phase 18)

## Typography

| Role | Size | Weight | Line Height | Font Family |
|------|------|--------|-------------|-------------|
| Body / Nav links / Footer links | 14px | 400 (regular) | 1.5 | Inter (--font-sans) |
| Brand wordmark | 20px | 700 (bold) | 1.2 | Space Grotesk (--font-display) |
| Footer heading / Instrument group label | 14px | 700 (bold) | 1.4 | Inter (--font-sans) |

Declared sizes: 14px, 20px (2 sizes). Declared weights: 400 (regular), 700 (bold) (2 weights).

Revision note: Footer links unified from 13px to 14px to eliminate a 1px size gap that provided no meaningful visual hierarchy. Footer heading weight changed from 600 (semibold) to 700 (bold) to stay within the 2-weight maximum. The footer heading and brand wordmark now share the same weight but are differentiated by font family (Inter vs Space Grotesk) and size (14px vs 20px).

Source: D-02 (Space Grotesk wordmark), existing nav.tsx (14px/text-sm links), Phase 19 type system

## Color

| Role | Token | OKLCH Value | Usage |
|------|-------|-------------|-------|
| Dominant (60%) | --color-bg | oklch(0.12 0.01 85) | Page background, main content area |
| Secondary (30%) | --color-surface | oklch(0.16 0.01 85) | Nav bar background, footer background |
| Nav surface | --color-surface-raised | oklch(0.20 0.015 85) | Nav bar elevated surface (D-01: raised elevation) |
| Accent (10%) — Evolver | --color-accent | oklch(0.85 0.18 105) | Active nav indicator bar, footer accent links, interactive highlights |
| Accent (10%) — Cascadia | --color-accent (overridden) | oklch(0.75 0.12 185) | Same elements as Evolver accent, warm teal variant |
| Border | --color-border | oklch(0.25 0.01 85) | Nav bottom border, footer top border |
| Muted text | --color-muted | oklch(0.58 0.01 85) | Inactive nav links, footer secondary text |
| Destructive | --color-destructive | oklch(0.55 0.2 25) | Not used this phase (no destructive actions) |

Accent reserved for:
- Active page indicator bar (2-3px bottom border on active nav link) — D-04
- Footer instrument quick-link hover state
- Demo mode badge background
- Callout borders that inherit accent (existing, unchanged)

Per-instrument accent override (D-10, D-11):
- `[data-instrument="evolver"]` keeps default `--color-accent: oklch(0.85 0.18 105)` (warm yellow-green)
- `[data-instrument="cascadia"]` overrides `--color-accent: oklch(0.75 0.12 185)` (warm teal)
- Cascadia teal value chosen for WCAG AA contrast (4.5:1) against --color-surface-raised background

Source: D-10, D-11, D-12, globals.css token layer (Phase 18)

## Responsive Breakpoints

| Breakpoint | Behavior |
|------------|----------|
| < 768px (mobile) | Hamburger nav, stacked footer columns, full-width page shells with lg horizontal padding |
| >= 768px (desktop) | Full horizontal nav links, two-column footer, page shells at declared max-widths |

No intermediate breakpoint needed for this phase. Medium shell width (960px) deferred unless module list pages demonstrate need during implementation.

---

## Accessibility Requirements

| Requirement | Implementation |
|-------------|----------------|
| Nav active state | aria-current="page" on active link |
| Mobile menu | aria-expanded on hamburger button, aria-hidden on menu panel when closed |
| Focus management | Focus trapped in mobile menu when open; Escape key closes menu |
| Color contrast | All text/background combinations must meet WCAG AA 4.5:1. Cascadia teal accent verified against raised surface |
| Reduced motion | Mobile menu: no slide animation when prefers-reduced-motion is enabled; instant show/hide |
| Skip navigation | Add `<a href="#main-content" class="sr-only focus:not-sr-only">Skip to content</a>` before nav |
