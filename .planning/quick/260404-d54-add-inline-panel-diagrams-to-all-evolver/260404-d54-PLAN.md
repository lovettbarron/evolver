Key reference for all tasks — the prototype session 05 demonstrates:
1. Panel after Setup section showing initial patch state (zoomed to oscillators)
2. Panel after Exercise 3's parameter list showing the culminating patch (zoomed to oscillators + envelope3)
3. Panels placed RIGHT BEFORE "Play a note" or "you should hear" lines
4. 1-2 panels per exercise maximum — don't overuse

Panel marker syntax:
```html
<div data-evolver-panel data-knobs="id:value,id:value" data-highlights="id:color,id:color" data-sections="section1,section2"></div>
```

Auto-zoom: when data-sections is set, panel auto-zooms to those sections.
Use data-zoom="false" to show full panel with highlights (good for multi-section patches).

Available section names: envelope3, lfos, sequencer, lcd, misc-params, modulators, transpose, oscillators, noise, ext-in, filter, amp, hp-filter, feedback, distortion, delay, output

Highlight colors: `blue` (informational/context) and `amber` (active/key controls)

Knob values are MIDI 0-127. Common mappings:
- Frequency C0=0, C1=12, C2=24, C3=36, C4=48
- Percentages roughly: 0=0, 64=50%, 127=100%
- Shape: Saw=0, Tri=32, P-50=64 (pulse 50%)
- For exact values, use the number stated in the session text if it's 0-127
</context>
